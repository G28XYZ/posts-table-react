## Установка

#### Клонировать репозиторий

`git clone https://github.com/G28XYZ/posts-table-react.git`

#### Перейти в каталог

`cd ./posts-table-react`

#### Установите зависимости

`nmp i`

#### Запустите приложение

`npm start`

#### Перейдите по адресу

`http://localhost:300`
